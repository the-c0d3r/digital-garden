# Complex system
topics: [[01-Project/012-Efforts/ongoing/Knowledge Management\|Knowledge Management]]

“A complex system that works is invariably found to have evolved from a simple system that worked. A complex system designed from scratch never works and cannot be patched up to make it work. You have to start over, beginning with a working simple system. – John Gall

[[Garden/knowledge-base/First Principles\|First Principles]] can help with building complex systems. The best type of system is an [[Garden/knowledge-base/Emergent System\|Emergent System]].
