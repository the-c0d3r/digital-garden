# Emergent System
topic: [[01-Project/012-Efforts/ongoing/Knowledge Management\|Knowledge Management]]

This is a type of system that "emerged" as you use your existing system and it improves based on your own modification, based on what works and what didn't. This is a good way to counter [[03-Resource/knowledge/Complex system\|Complex system]].

This means you can design the system from ground up as perfect as you want but over the course of the usage of the system, you will start making changes and what you have left with is the **Emergent System**.

This is a concept from [[03-Resource/knowledge/Molecular Notes\|Molecular Notes]]. This is also relevant with the [[10-Inbox/Second System Effect\|Second System Effect]], whereby the second time you build the system, you might have over engineered and lead to bad design, instead of following the emergent system.
