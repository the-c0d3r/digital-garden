# browser fingerprinting

Information on how the browser fingerprinting works, and maybe also how to protect yourself.

1. use browsers designed for privacy.
    1. don't use the browser created by the world's biggest advertising company (Google Chrome)
2. configure the settings to make it hard to track
3. install browser extensions
    1. ublock origin
    2. privacy badger
