# Second System Effect

topics: [[03-Resource/knowledge/Philosophy\|Philosophy]]

The first system is conservative as you don't know what you're doing. But the second system is when you start from scratch again, but over engineering all the features in the design, leading to failure. This leads to [[Garden/knowledge-base/Complex system\|Complex system]].
