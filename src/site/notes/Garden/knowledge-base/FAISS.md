# [[Garden/knowledge-base/FAISS\|FAISS]]

topics: [[02-Area/programming/AI/AI\|AI]]

source: [GitHub - facebookresearch/faiss: A library for efficient similarity search and clustering of dense vectors.](https://github.com/facebookresearch/faiss)

Faiss is a library for efficient similarity search and clustering of dense vectors. It contains algorithms that search in sets of vectors of any size, up to ones that possibly do not fit in RAM. It also contains supporting code for evaluation and parameter tuning.

Alternative: [Unum · USearch 0.22.3 documentation](https://unum-cloud.github.io/usearch/)
